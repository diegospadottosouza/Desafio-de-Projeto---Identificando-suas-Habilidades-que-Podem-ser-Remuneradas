# 💼 IA - Identificando suas Habilidades que Podem ser Remuneradas

Projeto desenvolvido como parte do **Desafio de Projeto da DIO - Inteligência Artificial na Prática**.  
O objetivo é ajudar usuários a reconhecer competências que podem ser monetizadas, indicando áreas de atuação, oportunidades de mercado e caminhos práticos para transformar habilidades em renda.

---

## 🚀 Funcionalidades
- Análise de habilidades informadas pelo usuário
- Sugestão de áreas de atuação e oportunidades de monetização
- Orientação prática para freelancing, consultoria e mercado digital
- Interface simples e interativa com IA

---

## 🛠️ Tecnologias Utilizadas
- Python 3.10+
- Bibliotecas: `streamlit`, `transformers`, `pandas`
- Engenharia de Prompt para IA Generativa
- GitHub Actions para automação

---

## 📂 Estrutura
- `src/` → Código principal da IA
- `data/` → Exemplos de habilidades e áreas de monetização
- `docs/` → Documentação adicional
- `requirements.txt` → Dependências do projeto

---
