# Desafio-de-Projeto---Identificando-suas-Habilidades-que-Podem-ser-Remuneradas
IA Habilidades Remuneradas é um projeto que analisa competências do usuário e sugere quais podem ser monetizadas. Utiliza inteligência artificial para identificar áreas de atuação, oportunidades de mercado e caminhos práticos para transformar habilidades em renda.
